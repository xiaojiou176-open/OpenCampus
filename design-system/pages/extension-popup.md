# Extension Popup Override

## Purpose

Popup 是 quick pulse，不是操作台，也不是导出仓库。

## Override Rules

1. 第一块必须是 `Pulse summary`
   - `Open assignments`
   - `Due soon`
   - `Unseen updates`
   - `Blocked sites`
2. 第二块是 `Fast actions`
   - `Open sidepanel`
   - `Sync current site`
   - `Export current view`
3. `Quick export` 作为第三块，不再独占页面
4. 小尺寸下只允许单列

## Copy Rules

- 用 `Quick pulse`
- 强调这是体温计，不是完整操作台
